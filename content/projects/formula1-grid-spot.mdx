# Formula 1 Grid Spot

A comprehensive full-stack web application that provides F1 enthusiasts with live race data, driver standings, session schedules, and an interactive community forum. Built with React/TypeScript frontend and Node.js/Express backend, it features real-time race tracking, user authentication, voting systems, and seamless integration with OpenF1 APIs to deliver an immersive racing experience for fans worldwide.

## Key Features

- Live race tracking and timing data
- Interactive session schedules and results
- Driver and constructor standings
- User authentication system
- Community discussion forum
- Real-time updates during race weekends

## Technologies Used

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Real-time: WebSockets
- APIs: OpenF1, Ergast API
